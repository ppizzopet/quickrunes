# QuickRunes

Rune Maker for LeagueOfLegends

## How to run

Install requirements

```bash
pip install -r requirements.txt
```

And then run it with

```bash
python main.py
```

## How to make runes

1. Run LeagueOfLegends.
2. Run the script.
